# Quant Portfolio Android

한국과 미국 주식/ETF를 계좌별, 자산별, 종목별로 관리하는 안드로이드 포트폴리오 앱입니다.

## 반영된 기능

- 계좌별 자산 현황
- 종목별 현재가 표시
- 현재가/환율 자동 갱신 버튼
- 표시 통화 선택
- 환율 변환
- 계좌별 손익 금액과 손익률
- 종목별 손익 금액과 손익률
- 수익은 빨강, 손실은 파랑 표시
- 목표비중 입력
- 리밸런싱 추가매수/축소 금액 계산
- AI 요청문 복사
- AI 추천 붙여넣기 적용
- CSV 가져오기/내보내기
- GitHub Actions APK 빌드

## APK 만들기

1. 저장소 상단의 Actions 탭을 엽니다.
2. Build Android APK 워크플로를 실행합니다.
3. 빌드가 끝나면 artifact `quant-portfolio-debug-apk`를 다운로드합니다.
4. 압축을 풀면 `app-debug.apk`가 들어 있습니다.

## 현재가와 환율

현재가는 Yahoo Finance chart API 형식의 티커를 사용합니다.

예시:

- 삼성전자: `005930.KS`
- SK하이닉스: `000660.KS`
- Apple: `AAPL`
- NVIDIA: `NVDA`
- S&P500 ETF: `VOO`

환율은 공개 환율 API를 통해 갱신합니다. 공개 API 기반이므로 실제 거래 체결 가격과 차이가 있을 수 있습니다.

## 주의

이 앱은 투자 관리와 리밸런싱 계산을 돕는 도구입니다. 실제 매수/매도 주문은 자동 실행하지 않습니다.
