# 자산 대시보드

미래에셋 보유자산을 기준으로 전체 자산, 계좌별 손익, 수익률을 확인하는 안드로이드 앱입니다.

## APK 만들기

GitHub 저장소 상단의 **Actions** 탭을 누른 뒤 **Build Android APK** 실행 결과를 엽니다.

빌드가 끝나면 아래 artifact를 다운로드합니다.

```text
mirae-asset-portfolio-debug-apk
```

압축을 풀면 `app-debug.apk`가 들어 있습니다.

## 현재 버전

현재는 빌드 성공을 우선한 최소 기능 버전입니다.

- 전체 평가금액 표시
- 전체 매입금액 표시
- 전체 손익 표시
- 전체 수익률 표시
- 손실 파란색
- 이익 빨간색
- 보유 종목 표

다음 단계에서 CSV 가져오기, 실제 차트, 현재가 조회 기능을 추가합니다.
